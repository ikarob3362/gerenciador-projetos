🔧 Configuração Inicial

Cole as credenciais do seu projeto Firebase para ativar o sistema.
Como obter:
1. Acesse console.firebase.google.com
2. Crie um projeto → Ative Authentication (Google) → Ative Realtime Database
3. Em Configurações do projeto → seus apps → Web → copie as credenciais
4. Na aba Realtime Database → Regras, cole:
{"rules":{".read":"auth!=null",".write":"auth!=null"}}
